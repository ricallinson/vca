# vca

A simple VLAN configuration analysis tool for Foundry/Brocade/Ruckus switches.

## Development Setup

Requires [Nodejs](https://nodejs.org/en/).

```
git clone git@github.com:ricallinson/vca.git
cd ./vca
npm i
```
